# Zarish – The Style Hub

This is a static website for Zarish – The Style Hub, a luxury jewelry brand specializing in handcrafted elegant pieces.

## Features

- Responsive design optimized for mobile, tablet, and desktop
- Modern typography using Playfair Display and Poppins fonts
- SEO optimized with Open Graph tags for social media
- Image optimization ready for WebP format
- Accessible design with proper ARIA labels
- Fast loading with optimized assets

## Pages

- **Home** (`index.html`) - Landing page with featured collections
- **About Us** (`about.html`) - Brand story and craftsmanship details
- **Collections** (`work.html`) - Jewelry collections showcase
- **Gallery** (`gallery.html`) - Photo gallery of jewelry pieces
- **Contact** (`contact.html`) - Contact information and inquiry form

## Collections Featured

1. **Eternal Sparkle** - Diamond Collection
